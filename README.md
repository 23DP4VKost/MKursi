# MKursi projekts

Šis projekts ir mācību darbs, kura mērķis ir izveidot pilna steka tīmekļa lietotni, izmantojot **Laravel** kā backend, **Vue.js** ar **Vuetify** kā frontend un **SQLite** kā datubāzi.

---

## Izmantotās tehnoloģijas

### Backend
- PHP 8.3
- Laravel 12
- SQLite

### Frontend
- Vue.js 3
- Vuetify 3
- Vite

# MKursi projekts

Šis projekts ir mācību darbs, kura mērķis ir izveidot pilna steka tīmekļa lietotni, izmantojot **Laravel** kā backend, **Vue.js** ar **Vuetify** kā frontend un **MySQL** kā datubāzi.

Projekta mērķis ir izveidot tīmekļa vietni matemātikas apguvei. Tēmas ir sadalītas matemātiskajās sadaļās (algebra, ģeometrija). Katra tēma ietver teoriju un praktiskos vingrinājumus, kas tiek glabāti datubāzē.

---

## Izmantotās tehnoloģijas

### Backend
- PHP 8.3
- Laravel 12
- MySQL

### Frontend
- Vue.js 3
- Vuetify 3
- Vite
